# ICS4U Culimating

## Project Overview

The project includes a tower defense game set in medieval times. The player will be a dragon and the main idea is to prevent enemy dragons from dealing damage to the objective (the castle). We chose this specific idea to integrate a lot of animation and collision as well as other topics learned in the class.

## Goals

- Fun Backstory
- Engaging Gameplayer
- Different Levels of challenge
